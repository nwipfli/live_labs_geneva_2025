# Instructions

1. Follow the [Launch from the Google Cloud console](https://docs.cloud.google.com/shell/docs/launching-cloud-shell#launch_from_the) to start a Cloud Shell from your console.
   
2. Clone the GitHub [repository](https://github.com/nwipfli/live_labs_geneva_2025.git):

`git clone https://github.com/nwipfli/live_labs_geneva_2025.git`

3. Change the directory and execute the script **prepare_project.sh**:

`cd live_labs_geneva_2025`\
`source prepare_project.sh`

4. Open [My notebooks](https://console.cloud.google.com/vertex-ai/colab/notebooks) in **Vertex AI Colab Enterprise** and follow the instructions [Import a notebook](https://docs.cloud.google.com/vertex-ai/docs/colab/create-console-quickstart#import) to import your first notebook from your Cloud Storage.

5. Open [BigQuery](https://console.cloud.google.com/bigquery).

6. Follow the instructions in [Connect to a non-default runtime](https://docs.cloud.google.com/bigquery/docs/create-notebooks#connect_to_a_non-default_runtime) to open your notebook and connect to your runtime ***standard-runtime***.

7. Follow the instructions in the notebook.
